# Task 3 – Exploratory Data Analysis (EDA)

 Objective
The goal of this task is to perform Exploratory Data Analysis (EDA) to understand
patterns, trends, and relationships in the dataset using visualizations.

---

Dataset
- Netflix Movies and TV Shows Dataset

---

 Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Google Colab

---
 EDA Steps
1. Dataset overview and structure analysis
2. Missing value analysis
3. Distribution of numerical features
4. Categorical feature analysis
5. Outlier detection using box plots
6. Correlation analysis using heatmap
7. Insights and summary

---

Repository Contents
- `Task-3.ipynb` – EDA notebook
- `netflix_titles.csv` – Dataset
- `README.md` – Task explanation

---

 Outcome
This task helped in understanding data patterns, feature relationships,
and the importance of visualization in data analysis.
